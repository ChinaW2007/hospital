## 一、系统组成

本系统包含以下四个子系统：

| 子系统 | 技术栈 | 端口 | 功能定位 |
|--------|--------|------|----------|
| HIS Server | Node.js + TypeScript + Express + MySQL | 3001 | HIS系统后端服务 |
| HIS Client | React + TypeScript + Vite | 3002 | HIS系统前端（医生/药师操作界面） |
| Hospital Back | Python + FastAPI + SQLite | 8000 | 药房智能系统后端服务 |
